# TP 1: Estructura HTML5 - Curriculum Vitae
## Descripción del Proyecto
Este proyecto consiste en la creación de un Curriculum Vitae digital utilizando etiquetas semánticas de HTML y estilos con CSS. El objetivo principal fue demostrar la transición de un diseño lineal a uno diagramado mediante la propiedad Flexbox.

## Información de Desarrollo
* Editor de código: Visual Studio Code
* Tamaño de pantalla de trabajo: 1280x720px

## Estructura del Proyecto
A continuación se detalla la organización de los archivos en el repositorio:

* img/: Carpeta que contiene los recursos visuales, incluyendo los logos de redes sociales.
* fuentes/: Carpeta destinada a los archivos de tipografía utilizados para personalizar el diseño del CV.
* index.html: Archivo principal que contiene la estructura semántica del Curriculum Vitae.
* estilos.css: Hoja de estilos externa donde se define la apariencia y la diagramación.
* .git/: Carpeta de control de versiones (invisible en el explorador de archivos estándar, pero presente en el repositorio).
